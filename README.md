# Bacterial Growth Segmentation using Multi-Threshold Fusion

This project performs:

- Image preprocessing
- Multi-threshold segmentation (Otsu + Adaptive + Manual)
- Morphological refinement
- Colony size analysis graphs
- Time-series bacterial growth curve

## Features
- Dropdown-based segmentation mode
- Automatic colony detection
- Growth visualization
- Lab-style analysis output

## Run

```bash
pip install -r requirements.txt
python main.py
